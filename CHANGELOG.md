# 开发日志

## [Unreleased] - 2026-07-20

### 新增

- 客户个人详情页，支持修改姓名和手机号、复制手机号、查看修改记录和管理客户存款。
- 产品管理，支持按银行维护产品，并为同一产品配置多个按日期生效的利率版本。
- 银行和产品可编辑下拉框：既可从目录选择，也可手工填写未收录名称。
- 产品候选根据银行和存入日期展示适用利率，目录产品使用强调色区分。
- 存款统计四种生命周期状态钻取，以及银行、产品统计明细。
- 存款详情中的续期、停止续期和编辑操作。
- 中文日期选择器。

### 调整

- 消息模板入口移入设置，底部导航精简为“首页、客户、新增、设置”。
- 产品目录和多版本利率纳入备份与恢复。
- 统计状态查询改为强类型对象，移除内部字符串协议。
- 客户修改记录显示中文字段名，并用颜色区分修改前后的值。

### 修复

- 修复 Android 通知初始化、权限重试和系统通知设置跳转。
- 修复 Android Release 资源压缩移除通知图标，导致通知初始化和首次权限申请失败的问题。
- 修复从客户详情新增存款时表单缺少 Material 页面容器，以及保存后未返回客户详情的问题。
- 修复续期后客户页面数据需要重启应用才刷新的问题。
- 修复复制提示和客户编辑对话框中的 Flutter 生命周期断言错误。
- 修复停止续期后提醒和操作状态未及时刷新的问题。
- 修复存款表单提示变化时文本字段状态被错误复用的问题。

### 验证

- `flutter analyze --no-pub`：通过，零问题。
- `flutter test --no-pub`：267 项全部通过。
- Android Debug APK：构建成功，并由用户完成当前版本手机主要流程验收。
- 首个正式版源提交为 `ec757ac`；签名 APK 和验收记录已完成，Tag 与 GitHub Release 待单独确认。
