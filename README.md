# 存款续期管理

一款离线优先的个人存款管理工具，用于记录客户与存款信息、跟踪到期和续期状态，并在 Android 设备上生成本地提醒。

当前版本：`1.0.0+2`

## 主要功能

- 首页按今日、三天内、本周内和到期待处理展示存款提醒。
- 从客户卡片进入个人详情，编辑姓名或手机号，并可一键复制手机号。
- 管理客户、存款及完整续期链，区分生效中、已续期、已停止和已到期状态。
- 在首页或存款详情中续期、停止续期和编辑活动存款。
- 生成可编辑的续期提示语并复制到剪贴板。
- 支持手工新增、离线文本识别和 `.xlsx` Excel 批量导入。
- 支持导入前预览、字段修正、重复数据处理和显式确认。
- 按银行管理产品，并为同一产品维护多个按日期生效的利率版本。
- 银行和产品使用可编辑下拉框，可选择已有目录项或手工填写。
- 选择银行后，产品候选按存入日期展示适用利率，选择后自动填入并允许手动覆盖。
- 日期选择器使用中文界面，同时保留 `YYYY-MM-DD` 键盘录入。
- 在设置中管理消息模板并生成预览。
- 查看当前本金、客户数、续期次数、银行和产品统计，并钻取四种生命周期状态。
- 导出本地备份、预览恢复影响、恢复数据并创建本地快照。
- 在 Android 上检查通知权限和精确提醒能力，并重排到期提醒。

## 平台与数据

- **主要平台：** Android 10 及以上版本（最低 API 29）。
- **桌面端：** 支持构建 Windows x64 版本；移动通知相关能力仍以 Android 为主。
- **数据存储：** 客户、存款、产品利率版本、模板和设置保存在设备本地 SQLite 数据库中。
- **数据迁移：** 通过应用内备份导出和恢复完成，不依赖云端账号或在线服务。
- **文本识别：** 在本地完成，识别结果必须由用户确认后才会保存。

## 使用流程

1. 在“新增”中手工录入存款，或通过文字、Excel 导入。
2. 在“客户”中进入客户详情，维护资料并查看存款和续期历史。
3. 在“首页”处理即将到期或已经到期的存款。
4. 对活动存款执行续期、停止续期或编辑。
5. 在“设置”中管理产品、消息模板、统计、通知权限和备份。

底部导航固定为“首页、客户、新增、设置”四项。

## Android 通知说明

首次启动时，应用可能请求通知权限。部分 Android 设备还需要允许精确提醒，才能尽量按计划时间触发通知。

Android 系统和厂商省电策略可能造成提醒延迟。应用被强行停止后，系统不会继续恢复提醒；重新打开应用会再次检查能力并重排计划。

## 本地开发

### 环境要求

- Flutter stable
- Dart `3.12.2` 或兼容版本
- Android SDK，构建 Android 时需要 JDK 17

### 获取依赖并验证

```powershell
flutter pub get
flutter analyze
flutter test
```

### 构建 Android Debug APK

```powershell
flutter build apk --debug
```

构建产物位于：

```text
build/app/outputs/flutter-apk/app-debug.apk
```

### 构建 Windows Release

```powershell
flutter build windows --release
```

Windows 版本必须连同 `Release` 目录内的 DLL 和 `data` 目录一起使用，不能只复制
`.exe` 文件。构建产物位于：

```text
build/windows/x64/runner/Release/
```

### 运行应用

```powershell
flutter devices
flutter run -d <device-id>
```

不要在公开日志或 Issue 中提交客户数据、备份文件、设备标识或其他敏感信息。

## 项目结构

```text
lib/
├── app/                 # 路由、主题、响应式导航和依赖绑定
├── core/                # SQLite、通知、备份及通用基础能力
└── features/
    ├── customers/       # 客户目录和存款续期链
    ├── dashboard/       # 首页提醒和快捷操作
    ├── deposits/        # 存款表单及续期工作流
    ├── excel_import/    # Excel 批量导入
    ├── settings/        # 统计、备份和通知设置入口
    ├── statistics/      # 存款统计
    ├── templates/       # 提示语模板
    └── text_import/     # 离线文本识别导入
```

## 已知限制

- 当前不提供云同步、多人协作或在线账号系统。
- Android 通知触发时间受系统权限、强行停止和省电策略影响。
- 自动快照当前只能查看记录，尚未提供从快照列表直接恢复的入口。
- Android Release 使用本地 `key.properties` 和 keystore 签名；签名材料不提交仓库。
